화면 안내를 따라 5단계로 이용할 수 있습니다. 시작하기 전에 빈 용기를 노즐 아래에 놓아주세요.

이용 방법 보기
리필 전 확인
  • 빈 용기를 안정적으로 놓아주세요.
  • 노즐이 용기 안쪽을 향하는지 확인하세요.
  • 리필 중에는 용기를 옮기지 마세요.

이용 방법

화면을 보며 순서대로 진행하세요

1

용기 준비

빈 용기를 노즐 아래에 놓아주세요

바닥이 평평한 곳에 용기를 놓고, 노즐 끝이 용기 안쪽을 향하게 맞춰주세요.

중요 저울이 있는 기기는 시작할 때 용기의 무게를 자동으로 맞춥니다. 시작 버튼을 누르기 전에 용기를 먼저 놓아주세요.

2

리필량 선택

화면 중앙의 리필량을 확인하세요

화면에 표시된 숫자가 리필할 양입니다. 조절 기능이 켜져 있으면 위로 밀어 늘리고, 아래로 밀어 줄일 수 있습니다.

참고 화면 아래에 조절 안내가 없다면 매장에서 정한 고정 용량으로 이용합니다.

3

시작 확인

화면을 탭하고 리필 시작을 눌러주세요

용량이 맞고 용기가 준비됐다면 화면을 한 번 탭하세요. 확인 화면이 나오면 큰 초록색 시작 버튼을 누릅니다.

주의 매장 설정에 따라 확인 화면 없이 바로 시작될 수 있습니다. 화면을 탭하기 전에 반드시 용기를 먼저 놓아주세요.

4

리필 진행

액체가 차오르는 동안 기다려주세요

화면의 색이 아래에서 위로 차오르며 진행 상태를 보여줍니다. 리필 중에는 용기와 노즐을 움직이지 마세요.

잠시 멈추기 화면을 한 번 탭
다시 시작하기 화면을 다시 탭
5

마무리

화면이 처음으로 돌아온 뒤 용기를 꺼내세요

펌프가 멈춘 뒤에도 노즐에 남은 액체가 잠시 떨어질 수 있습니다. 잠시 대기 또는 마무리 화면이 끝날 때까지 용기를 그대로 두세요.

완료 리필 기본 화면으로 돌아오고 액체가 더 이상 떨어지지 않으면 용기를 꺼냅니다.

넘치거나 이상할 때

화면을 탭한 뒤 빨간 정지를 눌러주세요

  1. 리필 중 화면을 한 번 탭합니다.
  2. 일시정지 화면의 빨간 정지 버튼을 누릅니다.
  3. 펌프와 액체가 완전히 멈춘 뒤 직원에게 알려주세요.

용기가 쓰러졌거나 누수가 생긴 경우 직접 기기를 옮기지 말고 직원에게 도움을 요청하세요.

자주 묻는 질문

이럴 때는 이렇게 하세요

리필량을 위아래로 바꿀 수 없어요.

매장에서 고정 용량으로 설정한 기기입니다. 화면에 보이는 용량으로 이용하거나 직원에게 문의하세요.

화면을 탭하자마자 바로 리필이 시작됐어요.

확인 화면을 사용하지 않는 설정입니다. 다음 이용부터는 용기를 먼저 놓고 용량을 확인한 뒤 화면을 탭하세요.

펌프가 멈췄는데 화면이 아직 진행 중이에요.

노즐에 남은 액체가 떨어지는 마무리 시간입니다. 화면이 처음으로 돌아올 때까지 용기를 그대로 두세요.

리필을 중간에 끝내고 싶어요.

화면을 한 번 탭해 일시정지한 뒤 빨간 정지 버튼을 누르세요.

문제가 생기면 즉시 정지하고 가까운 직원에게 알려주세요.
